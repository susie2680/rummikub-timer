# 拉密计时器

手机上使用的拉密游戏计时器。

## 功能

- 开始前选择 2-4 位玩家
- 每位玩家可输入名字并选择内置头像
- 按玩家顺序循环计时，每人 1 分钟
- 玩家出牌结束后点击大按钮，自动切换下一位
- 超时后提示并自动切换下一位
- 最后 10 秒有提示音
- 超时语音提示“超时，罚牌2张，到下一个玩家”并播报下一位玩家名字
- 手动结束语音提示“到下一个玩家”并播报下一位玩家名字
- 点击头像会自动把玩家名字改成对应称呼
- 右上角支持暂停、重新开始和换玩家

## 本地预览

```bash
npx serve .
```

也可以直接打开 `index.html`。
